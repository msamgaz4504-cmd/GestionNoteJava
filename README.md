# GestionNotes

Mini-projet Java pour gérer les notes d'une classe.

## Description
Ce projet permet :
- de saisir les notes de 5 étudiants
- de calculer la moyenne
- de trouver la meilleure note
- de compter le nombre d'étudiants admis
- d'afficher les mentions des étudiants

## Exécution
1. Ouvrir le projet dans IntelliJ IDEA
2. Exécuter le fichier `GestionNotes.java`
3. Suivre les instructions dans la console
